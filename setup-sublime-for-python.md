# 设置 Sublime 开发环境

by Dan Bader, 

## 安装插件

* Package Control
* SideBarEnhancements
* All Autocomplete
* SublimeCodeIntel
* SublimeREPL
* GitGutter
* Pylinter

## 设定 `subl` 命令行

```sh
$ sudo ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin
```
